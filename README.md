# Matlab Games

## 简介

- 使用 MATLAB GUI 制作一个小游戏三合一程序
- 三人合作开发, 每人各出一个小游戏
- 菜单界面和2048小游戏使用 `App Designer` 构建开发

## 开发环境

- 系统: `Windows 7 SP1 64-bit`
- MATLAB版本: `R2019a Update5 64-bit(win64)` (Professional License)

## 项目结构

> `.mlapp` 格式为 `App Designer` 默认生成的二进制文件

- [menu.mlapp](./menu.mlapp): 菜单界面
